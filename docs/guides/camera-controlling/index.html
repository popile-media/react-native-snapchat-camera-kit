Camera Controlling

Flip Camera​

Facing value could be front and back.

ref.current?.flipCamera().then((facing) => {
  // ..
});

Zoom​

Activating zoom feature:

 <CameraKit
  // ..
  zoom
 />

Value must be between 0 and 1, for an example 0.5.

// adjust zoom
ref.current?.zoom(amount);

Tap To Focus​

Activating tap to focus feature:

 <CameraKit
  // ..
  focus
 />
